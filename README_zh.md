# Endfield Tool —《明日方舟：终末地》生产计算工具

[English](./README.md)

## 项目简介

Endfield Tool 是一个面向《明日方舟：终末地》的生产规划和设施计算工具。  
它帮助玩家规划和优化资源生产：建造设施、制作物品、管理配方，并计算所需原材料及生产时间。

## 功能

- **设施管理**：查看和管理不同生产设施。
- **物品与配方**：浏览可制作的物品及其配方。
- **生产规划**：计算目标物品所需的原材料及生产链。
- **界面友好**：交互式界面，可调整生产数量并自动重新计算所需资源。

## 安装与运行

```bash
git clone https://github.com/JamboChen/endfield-tool.git
cd endfield-tool
pnpm install
pnpm dev
pnpm build
```

## 使用方法

1. 启动应用。
2. 选择要生产的目标物品。
3. 查看所需设施、原材料和生产时间。
4. 调整数量以重新计算资源。
